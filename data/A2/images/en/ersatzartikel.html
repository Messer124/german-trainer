Ersatzartikel (article as a noun substitute)

Masculine Neuter Feminine Plural
Nominativ einer / keiner eins / keins eine / keine welche / keine
Akkusativ einen / keinen eins / keins eine / keine welche / keine
Dativ einem / keinem einem / keinem einer / keiner welchen / keinen
Genitiv eines / keines eines / keines einer / keiner welcher / keiner

ein- / kein- — one / none — these pronouns can replace nouns that were mentioned earlier in context:

  • ○- In meinem Zimmer steht ein Tisch. — In my room there is a table. - Und in meinem steht auch einer! — And in mine there is one too!
  • ○- Hast du ein Messer? — Do you have a knife? - Nein, ich habe leider keins. — No, unfortunately I have none.

Attention! Possessive pronouns (mein, dein, sein) can also function as Ersatzartikel and behave like the indefinite article.
