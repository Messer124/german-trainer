Order of Objects in a Sentence

In a sentence that contains an Akkusativ object (answering "whom?" or "what?") and a Dativ object (answering "to whom?", "to what?"), the most common orders are:

  • If both objects are nouns, then Dativ comes first, followed by Akkusativ:

    Die Eltern kaufen dem Kind das Buch. — The parents buy the child a book.

  • If both objects are pronouns, then Akkusativ, comes first and Dativ follows:

    Die Eltern kaufen es ihm. — The parents buy it for him.

  • If one object is a noun and the other is a pronoun, the object expressed by a pronoun usually comes first, and the one expressed by a noun comes second:

    Die Eltern kaufen es dem Kind;

    Die Eltern kaufen ihm das Buch.
