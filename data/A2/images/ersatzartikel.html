Ersatzartikel (артикль - замена имени существительного)

Мужской род Средний род Женский род Множ. число
Nominativ einer / keiner eins / keins eine / keine welche / keine
Akkusativ einen / keinen eins / keins eine / keine welche / keine
Dativ einem / keinem einem / keinem einer / keiner welchen / keinen
Genitiv eines / keines eines / keines einer / keiner welcher / keiner

ein- / kein- — один / никакой — с помощью этих местоимений можно заменять имена существительные, которые появлялись в контексте ранее:

  • ○- In meinem Zimmer steht ein Tisch. — В моей комнате стоит стол. - Und in meinem steht auch einer! — И в моей тоже стоит один!
  • ○- Hast du ein Messer? — У тебя есть нож? - Nein, ich habe leider keins. — Нет, к сожалению, у меня нет (ни одного).

Внимание! Притяжательные местоимения (mein, dein, sein) могут также выступать в качестве Ersatzartikel и вести себя так же, как и неопределенный артикль.
